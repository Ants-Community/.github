# Contributing to ANTS

This is day zero. There is no network yet, and no code to run. The most valuable
thing you can contribute right now is not a pull request — it is a sharp,
specific disagreement.

## What helps most, in order

1. **Argue with the specification.** The RFCs in
   [`Ants-Community/ants`](https://github.com/Ants-Community/ants) under `spec/`
   are drafts written to be contested. If a mechanism is wrong, say where and
   why, concretely. "This breaks under X" beats "I don't like this."
2. **Open an issue, tagged.** Use the `rfc-000N` label for the RFC you are
   addressing and `design-question` if it is a design argument rather than a
   typo. The more concrete the critique, the faster it moves.
3. **Propose an amendment.** Substantive changes to an RFC are drafted as a
   patch against the file, with a rationale. Decisions are recorded in
   `spec/CHANGELOG.md` so the history of *why* is preserved, not just *what*.
4. **Fix typos and clarity.** Small PRs against the prose are merged quickly and
   are genuinely appreciated. Clear writing is part of the protocol.

## How discussion works

- Design questions live in issues first. If an issue resolves into a change, it
  becomes a PR or an amendment. If it needs wider argument, it moves to Matrix
  (`#ants-community:matrix.org`, room `#General`).
- We try to explain decisions, including the ones we got wrong. An amendment
  with an honest "we were wrong, here is why" is the normal shape of progress
  here, not an embarrassment.

## Licensing

By contributing you agree your contribution is released under the same terms as
the part of the repository you are touching: code and specifications under
Apache-2.0, the manifesto and the RFC prose under CC0. No CLA, no copyright
assignment. The work belongs to the commons.

## A note on pace

We are not in a hurry. This is a protocol meant to outlive its founders, so a
decision that is slow and defensible beats one that is fast and regretted. If a
response takes a few days, that is the system working, not failing.

The colony has no center. You are the center. Welcome.
