# Ants Community

> **An open protocol for collective intelligence.**
> *No tokens. No owners. Run a node.*

---

ANTS is a peer-to-peer network where every machine — a laptop in an office, a phone in a pocket, a GPU farm in a datacenter — both contributes to and draws from a shared, verifiable, sovereign artificial intelligence.

No single ant in a colony is intelligent. The colony is. We are building the same thing, with software.

We are not a company. We are not a token. We are not for sale.

We are a colony.

### Contributing

ANTS is at day zero.

The protocol is being written. The architecture is being debated. The colony does not yet exist.

If you build distributed systems, cryptographic protocols, or AI infrastructure — open an issue, propose a design, fork the spec.

If you do not, but you believe an artificial intelligence that belongs to no one is worth building — open an issue anyway.

No contribution is too small. No critique is unwelcome. No question is naive.
